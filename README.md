# Meal Planner App

A **full-stack meal planner** built with **FastAPI (Python)** backend, **React + TypeScript + Vite** frontend, and **JSON file storage**.  

The app allows you to:

- Store meals (name, calories, tags) in a JSON file
- Generate a **random weekly meal plan**
- View meals and weekly plan in a responsive frontend

---

## Tech Stack

- **Backend**: FastAPI, Python 3.12, Uvicorn  
- **Frontend**: React, TypeScript, Vite, Tailwind CSS  
- **Database**: JSON file (for prototyping; can be swapped for PostgreSQL later)  
- **CORS** enabled for local frontend access  

---

## Prerequisites

- **Python 3.12** ([download](https://www.python.org/downloads/release/python-3129/))  
- **Node.js 20+** ([download](https://nodejs.org/en/))  
- **npm** (comes with Node.js)
